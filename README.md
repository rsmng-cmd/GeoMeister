# GeoMeister - Geography Game

🌍 **Coğrafya bilginizi test edin!** Şehirleri haritada bulun, bayrakları tahmin edin ve çok oyunculu modda arkadaşlarınızla yarışın.

**Geliştirici:** Ares | **İletişim:** aresmungan@gmail.com

## Oyun Modları
- 🌍 Dünya — Dünya şehirlerini bulun
- 🇪🇺 Avrupa — Avrupa şehirlerini keşfedin
- 🇹🇷 Türkiye — Türkiye şehirlerini öğrenin
- 🚩 Bayrak Yarışı — Ülke bayraklarını tahmin edin
- ⚔️ Multiplayer — Gerçek zamanlı çok oyunculu

## Teknik
- Firebase (Firestore + Auth)
- Google AdMob
- PWA / Capacitor (Android)
- TR / EN dil desteği
